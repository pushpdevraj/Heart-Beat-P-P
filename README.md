

# Privacy Policy

Heart Beat Music Bot

Last Updated: December 2025

This Privacy Policy explains how Heart Beat Music Bot (“the Bot”) collects, uses, stores, and protects data when you use the Bot. By using the Bot, you agree to the practices described in this policy.



1. Information We Collect

Heart Beat collects only the minimum data required for proper functionality.

1.1 Automatically Collected Data

The Bot may store the following:

Discord User IDs

Discord Guild (Server) IDs

Discord Channel IDs

Voice Channel IDs

Bot configuration data (per-guild settings)

Music-related session data (queues, track identifiers, playback state)


This data is non-personal and cannot be used to identify users outside Discord.



2. How We Use Your Data

Collected data is used strictly to:

Provide music playback and voice channel functionality

Maintain per-server configurations

Ensure stability, performance, and security

Debug issues and improve features


No data is used for advertising, tracking, or profiling.



3. Data Storage & Security

Data is stored securely using industry-standard practices

Access is restricted to the Bot and its maintainers

Data is retained only as long as necessary for functionality


Despite reasonable safeguards, no system can guarantee absolute security.



4. Data Sharing

Heart Beat does not sell, trade, or share user data with third parties.

Data may be temporarily processed by third-party services strictly required for music playback or infrastructure (e.g., audio nodes or APIs). These services only receive what is necessary to function.



5. Third-Party Services

The Bot interacts with external services such as:

Discord API

Music streaming or audio processing services


Use of these services is governed by their respective privacy policies and terms.



6. User Rights

You have the right to:

Request removal of stored data related to your server

Remove the Bot from your server, which stops all future data collection


Data removal requests can be made via the official support or GitHub repository.



7. Children’s Privacy

The Bot is not intended for users under the age required by Discord’s Terms of Service. No data is knowingly collected from minors.


8. Policy Updates

This Privacy Policy may be updated at any time. Continued use of the Bot after changes indicates acceptance of the updated policy.


9. Contact Information

For privacy-related concerns or data removal requests:

Open an issue on the GitHub repository

Contact the maintainer through official support channels listed in the repository



By using Heart Beat Music Bot, you acknowledge that you have read and agreed to this Privacy Policy.
